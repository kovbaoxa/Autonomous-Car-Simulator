## Autonomous Car Simulator

### Dependencies

- Python 3
- numpy
- pygame

```
pip install -r requirements.txt
```

### Usage

#### Auto

Drive with your own algorithm which is implemented at `Brain.py`.
```
python main.py --auto
```

#### Not auto

Drive with Keyboard.

```
python main.py
```
