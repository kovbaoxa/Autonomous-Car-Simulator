## Autonomous Car Simulator

### Dependencies

- Python 3
- numpy (pip install numpy)
- pygame (pip install pygame)

### Usage

#### Auto

Drive with your own algorithm which is implemented at `Brain.py`.
```
python main.py --auto
```

#### Not auto

Drive with Keyboard.

```
python main.py
```
